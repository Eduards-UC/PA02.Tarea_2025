# PA02 – Tarea de Arquitectura del Computador 2025

## Descripción
Este repositorio contiene la resolución de los cuatro ejercicios en lenguaje ensamblador solicitados en el Producto Académico N.°2.

## Ejercicios
1. **Condición IF–THEN–ELSE:** comparación de dos variables y ajuste de valores.
2. **Bucle WHILE–DO:** cálculo de una serie Fibonacci básica.
3. **Máximo Común Divisor (MCD):** implementación del algoritmo de Euclides mediante restas.
4. **Intercambio de variables (SWAP):** aplicación de un ciclo y condición con intercambio.
